## <img src="https://img.shields.io/badge/-Featured%20Project-1a0b2e?style=flat-square"/>

<details open>
<summary><b>🔐 vericert — Digital Certificate Verification System</b></summary>
<br/>

**Overview**
A web-based system for verifying the authenticity of digital certificates, built to solve a real trust problem: how do you confirm a certificate wasn't forged or altered without manually contacting the issuer?

**Problem**
Physical and PDF certificates are easy to forge and hard to verify at scale — institutions and students both need a fast, trustworthy way to confirm a certificate is genuine.

**Solution**
A verification platform where certificates can be checked instantly via **QR code scanning**, backed by **hash integrity checks** to detect tampering, with dedicated dashboards for admins issuing certificates and students verifying theirs.

**Technology Stack**
`JavaScript` · HTML · CSS

**Core Features**
- QR code-based certificate validation
- Hash integrity verification to detect tampering
- Separate admin dashboard for certificate management
- Separate student dashboard for verification access

**Repository:** [github.com/aribashaikh02/vericert](https://github.com/aribashaikh02/vericert)

</details>
